## 订单编辑时库存账自动调整功能

### 问题背景
当在订单管理中修改手办的"定金"或"尾款"的金额或币种时，库存账需要同步记录成本调整，以保留审计轨迹并正确计算当前盈亏。

### 解决方案
在订单编辑逻辑中，当检测到订单金额发生变更时，自动创建一条 `quantity=0` 的 `adjust` 类型库存账记录来补差额。

### 实现逻辑

**记录特征：**
- 类型为 `adjust`（调整类型）
- 数量为 0（不增加库存数量）
- 金额为差额（正数表示成本增加，负数表示成本减少）
- 关联原订单 ID
- 备注包含调整原因和金额变化详情

**触发场景：**
1. 定金金额增加/减少
2. 尾款金额增加/减少
3. 定金币种变更
4. 尾款币种变更

**计算规则：**
所有币种统一转换为人民币计算差额，当新旧订单的人民币总金额差异大于允许误差时，自动生成调整记录。

**示例：**
- **尾款优惠**：原订单总价 586.96 CNY → 578.26 CNY，生成差额为 -8.70 的调整记录
- **追加定金**：原订单总价 500 CNY → 800 CNY，生成差额为 +300 的调整记录

### 实现效果

- 订单金额变更时自动创建库存账调整记录
- 支持定金和尾款的金额/币种变更追踪
- 保留完整的审计轨迹（资金账 + 库存账）
- 正确计算盈亏（调整记录参与成本计算）
- 不影响库存数量

## 修复库存账调整记录 price 计算逻辑

### 问题背景
当订单的定金或尾款发生变更时，库存账 `adjust` 类型记录的金额字段计算错误。之前存储的是差额，现调整为存储**该笔调整后的订单总成本**。

### 解决方案
修改库存账调整记录的金额计算逻辑：
- 金额字段存储的是调整后的订单总成本（定金人民币 + 尾款人民币）
- 而非仅存储差额

**计算规则：**
- 当定金发生变更时：新的总成本 = 新定金人民币 + 当前尾款人民币
- 当尾款发生变更时：新的总成本 = 当前定金人民币 + 新尾款人民币

### 实现效果

- 金额字段存储的是调整后的订单总成本
- 计算逻辑：定金 × 汇率 + 尾款 × 汇率
- 备注中保留变更字段的区分（定金/尾款）

## 优化平均入手价格计算逻辑

### 问题背景
之前的平均入手价格计算逻辑存在以下问题：
1. 已取消订单的尾款也被计入总成本，但实际上已取消订单只有定金是真实支付的
2. 计算平均价格时，分母包含了已取消订单的数量

### 解决方案
修改平均入手价格的计算规则：

**计算公式：**
平均入手价格 = (所有订单的实际支付金额总和) / (非已取消订单数量)

**计算规则：**
- 已取消订单的实际支付金额 = 定金（转换为人民币）
- 非取消订单的实际支付金额 = 定金 + 尾款（都转换为人民币）
- 数量分母：只统计非已取消订单的数量

**计算示例：**

某手办有 3 个订单：
- 订单1（已预订）：定金 500 CNY + 尾款 2000 CNY = 2500 CNY
- 订单2（已取消）：定金 300 CNY（尾款未付）
- 订单3（已预订）：定金 100 USD(700 CNY) + 尾款 200 USD(1400 CNY) = 2100 CNY

**优化前：** 总金额 4900 / 订单数 3 = 1633.33 CNY  
**优化后：** 总金额 4900 / 有效订单数 2 = 2450 CNY

### 实现效果

- 已取消订单只统计定金，不统计尾款
- 平均价格的分母只包含非已取消订单数量
- 更准确地反映实际投入成本
- 影响手办详情页、手办列表、资产看板等所有显示平均入手价格的地方

## 资产计算服务拆分（补充）- 市场基准服务

### 问题背景
继续采用企业级服务层架构，将塑料指数计算和跑赢大盘百分比计算两个方法从主资产计算服务拆分到独立的服务模块中。

### 解决方案
创建市场基准服务类，专门负责市场基准相关的计算逻辑，包括：
1. 塑料指数计算（市值加权复权指数）
2. 跑赢大盘百分比计算（与上证指数对比）

### 实现逻辑

**塑料指数计算：**
- 公式：塑料指数 = 基准日指数 × (当前总市值 / 基准日总市值)
- 基准日指数设定为 1000
- 基准日为最早购买手办的日期（开户首日）
- 基准日上证指数设定为 2900
- 新买入手办不改变指数，只增加成分股
- 卖出手办不影响指数，视为成分股剔除
- 再版冲击（复刻）通过市场价变化自然反映在指数中

**跑赢大盘百分比计算：**
- 塑料指数涨幅 = (当前塑料指数 - 基准日指数) / 基准日指数 × 100%
- 上证指数涨幅 = (当前上证指数 - 基准日上证指数) / 基准日上证指数 × 100%
- 跑赢大盘 = 塑料指数涨幅 - 上证指数涨幅

**服务拆分方式：**
- 主资产计算服务保持向后兼容，通过委托调用方式使用新的市场基准服务
- 市场基准服务独立封装，符合单一职责原则
